# Buddy-MyVoiceAssistant

This is a voice automated software. 

Currently it can perform following tasks :
  
  1. Launch certain softwares
  2. Set alarm 
  3. Read pdf
  4. Play games 
    a. Hangman
    b. Rock paper scissors
    c. Tic tac toe
